# Contact Form (HTML, CSS, PHP)

A simple web-based contact form that collects user information and processes it using PHP.

## 🚀 Features
- User-friendly form interface
- Input fields for name, email, and message
- Server-side processing using PHP
- Displays submitted data

## 🛠️ Technologies Used
- HTML
- CSS
- PHP

## ▶️ How to Run
1. Install XAMPP/WAMP
2. Place files in htdocs folder
3. Start Apache server
4. Open in browser:
   http://localhost/contact-form/index.html
